A class for the Bonn Shutter

Attributes¶

port_namestr

The name of port to which the Bonn shutter is connected to. For a windows machine it is a COM* port For a Linux machine it is /ttyUSB*

Baud_rateint

the baud rate of serial data communication

Bits_per_byteint

Number of data bits in byte data

Stop_bitsint

number of stop bits indicating end of a byte word.
